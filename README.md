# 🎂 Age Calculator

A clean, responsive age calculator web app that computes your exact age in years, months, and days from your date of birth — built with pure vanilla JavaScript.

---

## ✨ Features
- Date picker with future dates automatically disabled
- Calculates exact age in years, months and days
- Handles edge cases like month-end date differences
- Clean gradient UI with no frameworks or libraries

## 🛠️ Tech Stack
- HTML
- CSS (Gradient UI, Flexbox)
- Vanilla JavaScript

## ⚙️ How to Run
1. Clone the repo
git clone https://github.com/Somnath-001/age-calculator.git

2. Open index.html in your browser
No installation needed — pure HTML/CSS/JS.

## 🧠 How It Works
- Captures date of birth from date picker
- Gets today's date using JavaScript Date object
- Calculates difference in years, months and days
- Handles month boundary edge cases using getDaysInMonth()
- Displays result dynamically without page reload
